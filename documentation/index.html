HTML Structure


The html5 template uses Latest Bootstrap v3.3.4 with valid HTML5 tags. This theme is a responisve layout with 3 column Support column. All of the information in content area is nested within a class and comes with predefined classes.

CSS Files and Structure


Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is style.css which for this template and third one is responsive.css to control responsive layouts.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Jquery and javascript


This theme imports three types of Javascript files.

  • jQuery
  • Theme scripts (Theme.js)
  • scripts (script.js)
  • Some plugins

Google Map Settings


You have to change the latitude and longtitude value of the map element's data attribute . If you dont know how to get latitude and longitude value search on google or send us your location in google map,we will provide you the latitude and longtitude value or we will send you the complete map code as per your need. here is a screenshot that shows where exactly you have to change these values.


Those are required attribute which is listed below:

  unique id

  data-map-lat

  data-map-lng

  data-map-zoom

Contact Form Settings
